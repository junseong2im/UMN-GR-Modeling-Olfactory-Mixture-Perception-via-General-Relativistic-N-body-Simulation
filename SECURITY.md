# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly.

**Contact:** junseong2im@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Scope

This repository contains research code for the paper:

> **UMN: Universe Multi Neural Network for Olfactory Mixture Similarity Prediction via Differentiable N-body Simulation**

The code is provided for academic reproducibility and is not intended for production deployment.
